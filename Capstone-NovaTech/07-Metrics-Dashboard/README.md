## Dashboard Design Requirements

- **RAG status** for every metric (Red/Amber/Green)
- **Trend arrows** (↑ improving, ↓ declining, → stable)
- **3-quarter trend data** minimum (Q1, Q2, Q3, Q4)
- **"So what" narrative** for each metric (1 sentence explaining business significance)
- **Overall Security Posture Indicator** — weighted composite score
- Written for **non-technical board audience**
