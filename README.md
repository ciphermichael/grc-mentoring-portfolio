## 📋 About This Repository

This repository is the official learning hub for the **6-Month Cybersecurity GRC Mentorship Program**. It contains:

- ✅ **24 weekly assignments** with real-world scenarios
- ✅ **15 portfolio-grade GRC projects** ready for GitHub showcase
- ✅ **20+ professional templates** (risk registers, policies, audit checklists)
- ✅ **Example evidence files** in XLS and PPT format (what real GRC deliverables look like)
- ✅ **A complete capstone project** simulating an enterprise GRC engagement
- ✅ **Interview preparation** for every topic covered

## 📅 24-Week Program Overview

| Phase | Weeks | Focus | Key Projects |
|-------|-------|-------|-------------|
| **Phase 1: Foundations** | 1–4 | GRC fundamentals, risk basics, governance, compliance intro | GRC Landscape Report, Risk Register, Governance Framework |
| **Phase 2: Frameworks** | 5–10 | ISO 27001, NIST CSF, Policies, Vendor Risk, SOC 2, GDPR | ISO 27001 SoA, NIST CSF Assessment, Policy Library |
| **Phase 3: Audit & Testing** | 11–16 | Internal audit, control testing, BIA, IR, metrics, CIS | Internal Audit Report, Control Testing Workbook |
| **Phase 4: Advanced GRC** | 17–20 | Enterprise risk, FAIR model, cloud GRC, career prep | FAIR Analysis, Cloud Posture Assessment |
| **Capstone** | 21–24 | Complete enterprise GRC program for NovaTech Corp | Full GRC Program (all deliverables) |

## 🚀 Getting Started

### Week 1 Checklist

- [ ] Fork this repository to your GitHub account
- [ ] Set up your GitHub profile (photo, bio mentioning GRC specialization)
- [ ] Read `00-Program-Overview/curriculum.md`
- [ ] Complete the `Week-01` assignment
- [ ] Submit your first GitHub portfolio entry

### How to Submit Assignments

1. Complete the assignment in the relevant `Week-XX/` folder
2. Upload your deliverables to the appropriate subfolder
3. Update the `Week-XX/README.md` with your completion status
4. Tag your mentor for review using `@mentor-name`
